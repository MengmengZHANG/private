What is Week Report System?

This system is designed for:

  • employees to report what they have done every work day
  • employers / managers to review what their team members have done every work day
  • manager can see but not modify the report of its team member

How an employee can report their daily work?

At Home page, one calendar will be displayed. The user can click on the date it wants to report, and in the popup window report his work event

This event can be also be seen by the employee's manager, which is defined at Organization page

How an manager review its team member's report?

At Home page, at the right panel. It's showing the currently displayed calendar belongs to which user, by default it's the current logged in user.

A manager can expand / collapse the user tree to see its team members, and double click on it to show its content.

How to define the hierarchy of manager/employee?

For example, if you want to define that user1 is the manager of user2, and user2 is the manager of user3 and user4, you can:

  • Go to Organization page
  • Create one new record, with user name user1
  • Create another one new record, with user name user2, manager name user1
  • Create another one new record, with user name user3, manager name user2
  • Create another one new record, with user name user4, manager name user2
